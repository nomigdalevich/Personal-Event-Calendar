# Personal Event Calendar

JavaScript-based personal calendar application for managing events throughout the year.  
Users can add events by date and view them in a structured calendar layout.

## Features
- Add events to specific dates
- Display events by day and month
- Simple and clear calendar view
- Client-side logic using JavaScript

## Technologies
- JavaScript
- HTML
- CSS

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/nomigdalevich/Personal-Event-Calendar.git
   ```
2. Open the project folder.
3. Open the main HTML file in a web browser to view the application.
